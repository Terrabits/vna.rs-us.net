R&S Verify Cal

R&S Verify Cal verifies the accuracy of a calibration by performing measurements on known standards and comparing the result to acceptable limits*

Features:

  • Works with any user-defined standard
    • Acceptable limits are arbitrarily defined
  • Test Plans can be created by engineers, saved and then distributed in a production environment
  • Testing is guided by a graphical user interface
  • Pass/Fail results are clear

Requirements:

  • Rohde & Schwarz Vector Network Analyzer
  • Windpws PC (Optional)
  • Windows XP Service Pack 2 or newer

Installation:

  • Download the installer to your PC or the instrument
  • Run the installer, following the on-screen instructions

Instructions:

  • On a ZVA:
    Press the "System Config" hard key twice, choose the "External Tools" soft key and select "Verify Cal"
  • On a ZNB:
    Press the "Applic(ations)" hard key, then select "Verify Cal"
  • From Windows:
    Navigate to Start-Programs-R&S Verify Cal

Downloads:

  • Application Card
  • Installer
  • Source code is available on request

*Important Note: The magnitude of the max and min limit files is absolute, but the phase is relative to the nominal value
