The python package rohdeschwarz provides a VNA instrument driver for python 2.7/3.

Install

As you would expect, you can install rohdeschwarz from the command line with pip: {% highlight ruby %} pip install rohdeschwarz {% endhighlight %}

Requirements

The pyvisa and numpy python packages must be installed.
There is a direct tcp socket option if you prefer to connect to the instrument without installing VISA on the host machine. However, the pyvisa package is still required at the moment.

Getting Started

The following code snippet will attempt to connect to an instrument via TCPIP at the address provided (in this example on the local system). {% highlight python %} from rohdeschwarz.instruments.vna import * vna = Vna() vna.open('TCPIP', '127.0.0.1') # # Continue with vna driver... # {% endhighlight %}
