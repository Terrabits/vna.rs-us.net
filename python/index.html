The python package rohdeschwarz provides a VNA instrument driver for python 2.7/3.

Requirements

The pyvisa and numpy python packages must be installed. Numpy is tricky to install because it includes python language extensions written in C that must be compiled for your specific platform. The easiest way to accommodate this is to find a precompiled version of python and numpy that work together. Download each, install python first and then install numpy. Then pip install rohdeschwarz.
There is a direct tcp socket option if you prefer to connect to the instrument without installing VISA on the host machine. However, the pyvisa package is still required at the moment, even if it is not used.

Install

As you would expect, you can install rohdeschwarz from the command line with pip:

{% highlight ruby %} pip install --pre rohdeschwarz {% endhighlight %}

The --pre flag indicates that you are okay with a version of rohdeschwarz that is "pre-release" (beta).

Getting Started

The following code snippet will attempt to connect to an instrument via VISA and GPIB at the address provided.

{% highlight python %} from rohdeschwarz.instruments.vna import Vna vna = Vna() vna.open('GPIB', '17') vna.open_log('path/to/scpi/log.txt') {% endhighlight %}

Alternatively, to connect via a direct TCP socket (no VISA installation required):

{% highlight python %} from rohdeschwarz.instruments.vna import Vna vna = Vna() vna.open_tcp('192.168.1.101') vna.open_log('path/to/scpi/log.txt') {% endhighlight %}
