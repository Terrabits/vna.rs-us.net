R&S User Defined Ports

R&S User Defined Ports allows the definition of ports with an arbitrary source, reference receiver and measurement receiver. R&S User Defined Ports also includes setup wizards for several high power measurements.

Features:

  • Manual creation of user-defined ports
  • Wizards for setting up:
    • High power DUT measurements
    • Automatic Level Control (ALC) for precise power measurements using external power meters

Requirements:

  • Rohde & Schwarz Vector Network Analyzer
  • At least four ports or direct port access
  • Windows XP Service Pack 2 or newer

Installation:

  • Upload the installer to the instrument
  • Run the installer, following the on-screen instructions

Instructions:

  • On a ZVA:
    Press the "System Config" hard key twice, choose the "External Tools" soft key and select "User Defined Port"
  • On a ZNB:
    Press the "Applic(ations)" hard key, then select "User Defined Port"
  • From Windows:
    Navigate to Start-Programs-R&S User Defined Ports
