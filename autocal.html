TOSM with Autocal

This application uses a Rohde & Schwarz Autocal Unit for the Open-Short-Match (OSM) standards in a manual TOSM or UOSM calibration. An external thru standard is required.

Features:

  • Manual calibration with the convenience of a R&S Autocal unit
  • Improved accuracy of a TOSM calibration

Requirements:

  • Rohde & Schwarz Vector Network Analyzer
  • Rohde & Schwarz Autocal Unit
  • Manual thru standard

Installation:

  • Upload the installer to the instrument
  • Run the installer, following the on-screen instructions

  • On a ZVA:
    Press the "System Config" hard key twice, choose the "External Tools" soft key and select "TOSM with Autocal"
  • On a ZNB:
    Press the "Applic(ations)" hard key, then select "TOSM with Autocal"
  • From Windows:
    Navigate to Start-Programs-R&S TOSM with Autocal
