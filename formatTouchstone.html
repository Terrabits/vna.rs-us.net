R&S Format Touchstone File

R&S Format Touchstone File reformats a given touchstone file. This is useful when the parsers for other software do not implement the touchstone file format correctly. Format Touchstone File can also be used to convert a touchstone file for CSV format.

Features:

  • Remove excess whitespace
  • Remove comments
  • Convert to CSV file format
  • Format files for Windows compatibility

Requirements:

  • Windows XP Service Pack 2 or newer

Installation:

  • Can be installed on- or off-instrument
  • Run the installer, following the on-screen instructions

Instructions:

  • On a ZVA:
    Press the "System Config" hard key twice, choose the "External Tools" soft key and select "Format SNP"
  • On a ZNB:
    Press the "Applic(ations)" hard key, then select "Format SNP"
  • From Windows:
    Navigate to Start-Programs-R&S Format Touchstone File
