Features:

Requirements:

  • Rohde & Schwarz Vector Network Analyzer
  • Windpws PC (Optional)
  • Windows XP Service Pack 2 or newer

Installation:

  • Upload the installer to the instrument
  • Run the installer, following the on-screen instructions

  • On a ZVA:
    Press the "System Config" hard key twice, choose the "External Tools" soft key and select "Display Offsets"
  • On a ZNB:
    Press the "Applic(ations)" hard key, then select "Display Offsets"
  • From Windows:
    Navigate to Start-Programs-R&S Display Offsets
