R&S Display Offsets

Display Offsets is a utility for Rohde & Schwarz Vector Network Analyzers that displays the relevant delay offsets in the title bar of each window.

Features:

  • Only the delay offsets relevant to the traces are displayed
  • Zero-valued offsets are not displayed
  • Offsets are displayed in engineering notation

Requirements:

  • Rohde & Schwarz Vector Network Analyzer
  • Windows XP Service Pack 2 or newer

Installation:

  • Upload the installer to the instrument
  • Run the installer, following the on-screen instructions

Instructions:

  • On a ZVA:
    Press the "System Config" hard key twice, choose the "External Tools" soft key and select "Display Offsets"
  • On a ZNB:
    Press the "Applic(ations)" hard key, then select "Display Offsets"
  • From Windows:
    Navigate to Start-Programs-R&S Display Offsets
